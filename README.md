# SmartHire AI - Resume Analyzer

SmartHire AI is an AI-powered Resume Analyzer and Job Matching System designed to help recruiters and job seekers evaluate resume compatibility with job requirements.

## Features
- Upload and analyze resumes
- Compare resumes with job descriptions
- Generate resume-job match scores
- Detect relevant technical and professional skills
- Provide candidate suitability recommendations
- Clean and responsive user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
1. Upload a resume.
2. Paste the job description.
3. Click "Analyze".
4. View the match score, detected skills, and recommendation.

## Future Enhancements
- PDF/DOCX resume parsing
- AI-based skill extraction
- ATS score calculation
- Resume improvement suggestions
- Integration with LLM APIs (OpenAI/Gemini)
- Real-time job recommendations
